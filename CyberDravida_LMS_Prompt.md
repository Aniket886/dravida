### **Prompt Title: Build a Complete Course Selling LMS Platform for Cyber Dravida**

I need you to build a professional Learning Management System (LMS) platform for my cybersecurity education business called **Cyber Dravida**. This platform should allow me to sell online courses, manage student enrollments, and provide a seamless learning experience. Here are the complete requirements:

#### **1. BRANDING & VISUAL DESIGN**
- Organization Name: **Cyber Dravida**
- Primary Color Scheme: Professional dark theme (deep blue/teal with cybersecurity-focused aesthetics)
- Logo Area: Include a prominent space for the Cyber Dravida logo at the top
- Fonts: Modern, clean, professional fonts (sans-serif)
- Responsive Design: Must work perfectly on desktop, tablet, and mobile devices

#### **2. CORE PAGES & FUNCTIONALITY**

**A. Homepage/Landing Page**
- Eye-catching hero section with "Cyber Dravida" tagline
- Welcome message about cybersecurity education and courses
- Course showcase section displaying 3-4 featured courses with thumbnails
- Brief company description/about section
- Call-to-action buttons for "Browse Courses" and "Explore Training"
- Footer with contact information, social media links, and copyright

**B. Courses Catalog/Discovery Page**
- Grid view of all available courses (minimum 12 slots for courses)
- Each course card should display:
  - Course thumbnail image
  - Course title
  - Brief description (2-3 lines)
  - Course level (Beginner, Intermediate, Advanced)
  - Duration (hours of training)
  - Price (showing both discounted and original price if applicable)
  - Student count/enrollment statistics
  - Star rating (1-5 stars)
  - "Enroll Now" or "Add to Cart" button
- Filter functionality by:
  - Course level (Beginner, Intermediate, Advanced)
  - Price range (slider)
  - Category (OSINT, Ethical Hacking, Network Security, Malware Analysis, etc.)
- Search bar to find courses by keyword/name
- Sorting options (Most Popular, Newest, Price Low-High, Price High-Low, Highest Rated)

**C. Course Details Page**
- Large course thumbnail/banner image at the top
- Course title, instructor name, and rating with number of reviews
- Quick stats: (Enrolled Students, Course Duration, Skill Level, Last Updated)
- Detailed course description/overview
- Course curriculum/modules section showing:
  - Module name
  - Number of lessons in each module
  - Total duration
  - Expandable view to see lesson titles within each module
- Instructor profile card showing:
  - Instructor photo
  - Name and title
  - Bio/expertise
  - Other courses by instructor
- Course requirements section (prerequisites if any)
- Learning outcomes/what you'll learn (bullet points)
- Pricing section with:
  - Course price prominently displayed
  - Discount information (if applicable)
  - "Enroll Now" button (large, prominent)
  - Money-back guarantee notice if applicable
- Student reviews section (showing 3-5 recent reviews with ratings)
- Related courses section (similar courses)

**D. Student Dashboard (Authenticated Users)**
- Dashboard home showing:
  - Welcome message with student's name
  - Progress summary widget
  - Enrolled courses section
- My Courses page showing:
  - Cards for each enrolled course
  - Progress bar showing completion percentage
  - Last accessed date
  - "Continue Learning" button for each course
  - Quick stats (modules completed, time spent, certificates earned)
- Learning Progress Tracker:
  - Overall learning analytics
  - Total hours spent on platform
  - Courses completed
  - Certificates earned
  - Course-by-course progress breakdown
- Wishlist/Saved Courses section
- Payment History & Receipts showing:
  - Past purchases
  - Transaction dates
  - Order IDs
  - Download receipt option
- User Profile/Account Settings:
  - Profile information (name, email, phone, location)
  - Change password
  - Notification preferences
  - Download certificates earned
  - Account deletion option

**E. Student Course Player/Learning Page**
- Left sidebar showing course structure:
  - Module list (expandable)
  - Lesson list within each module
  - Current lesson highlight
  - Mark lesson as complete checkbox
- Main content area showing:
  - Video player for video lessons (with pause, play, fullscreen, playback speed)
  - Lesson title
  - Lesson description/notes
  - Downloadable resources (PDFs, code files, etc.)
  - Lesson transcript (if applicable)
- Right sidebar or bottom section showing:
  - "Mark Complete" button
  - Navigation (Previous/Next lesson buttons)
  - Progress bar for current course
  - Time spent on current lesson tracker
- Quiz/Assessment section within lessons:
  - Multiple choice questions
  - True/False questions
  - Submit button
  - Instant feedback showing correct/incorrect answers
  - Score percentage

**F. Shopping Cart & Checkout**
- Add courses to cart from catalog
- Cart summary showing:
  - Course name and price
  - Quantity (mostly 1 for courses)
  - Subtotal
  - Discount code input field
  - Tax calculation (if applicable)
  - Total price (prominent display)
- Remove from cart option
- Proceed to checkout button
- Checkout page with:
  - Billing information form (Name, Email, Phone, Address)
  - Payment method selection (Credit Card / Debit Card initially)
  - Stripe or PayPal integration for payment processing
  - Order review before final payment
  - Order confirmation page with:
    - Order ID
    - Course access details
    - Download invoice option
    - Button to go to dashboard/start learning

**G. User Authentication Pages**
- Sign Up/Registration page:
  - Email input
  - Full Name input
  - Phone number input (optional)
  - Password input with strength indicator
  - Confirm password input

  - Link to signup page
- Password Reset page:
  - Email input
  - Send reset link button
  - New password input (after clicking reset link)
  - Confirm password input
  - Reset password button
- User profile/avatar system with initials or profile picture

**H. Admin/Course Creator Dashboard (Optional but Recommended)**
- Admin login panel
- Courses Management:
  - List all courses
  - Add new course form
  - Edit course details
  - Delete courses
  - Manage course modules and lessons
  - Upload course materials
- Student Management:
  - View all enrolled students
  - Student progress tracking
  - Manage enrollments
  - Send notifications/announcements
- Analytics & Reports:
  - Total revenue generated
  - Total students
  - Course performance metrics
  - Popular courses
  - Student engagement analytics
  - Sales charts and graphs
- Payment Management:
  - View all transactions
  - Payout information
  - Revenue reports
  - Discount/coupon management

#### **3. KEY FEATURES**

**User Roles:**
- **Public Users**: Can browse catalog, view course details, no login required
- **Registered Students**: Can purchase courses, enroll, access learning materials, track progress
- **Instructors/Admins**: Can manage courses, students, and analytics

**Payment & Monetization:**
- Stripe or PayPal integration for course payments
- Support for multiple currencies (INR preferred for India market)
- Automatic invoice generation and download
- Support for discount codes/coupon codes
- Support for course bundling (buy multiple courses at discount)
- Pricing variations (Regular price, Promotional price, Early bird discount)

**Learning Management:**
- Course enrollment system (automatic on payment)
- Lesson completion tracking
- Progress percentage per course
- Quiz/assessment integration
- Certificate generation upon completion
- Time tracking (hours spent per lesson/course)
- Bookmark/save lessons for later

**Student Engagement:**
- Email notifications for new courses, updates, reminders
- Course announcements/notifications visible in student dashboard
- Discussion forums or comment sections (optional for Phase 1)
- Wishlist/Save for later functionality
- Course recommendations based on learning history
- Leaderboard showing top learners (optional)

**Content Management:**
- Support for multiple content types:
  - Video lessons (mp4, webm, YouTube embeds)
  - PDF documents
  - Code files/resources
  - Text-based lessons
  - Downloadable materials
- Lesson descriptions and notes
- Lesson sequencing and module organization
- Prerequisite system (optional)

**Security & Access Control:**
- Student authentication required for course access
- One-time purchase access (no subscriptions initially)
- Content protection (cannot download/view without enrollment)

**Notifications & Messaging:**
- Welcome email to new students

Create database tables/collections for:
- **Users** (id, email, password, name, phone, createdAt, role)
- **Courses** (id, title, description, price, level, duration, instructor, thumbnail, createdAt)
- **Modules** (id, courseId, title, order, content)
- **Lessons** (id, moduleId, title, content, videoUrl, resources, order)
- **Enrollments** (id, userId, courseId, enrolledAt, completionPercentage, status)
- **Payments** (id, userId, courseId, amount, paymentMethod, status, transactionId, createdAt)
- **Certificates** (id, userId, courseId, issuedAt, certificateNumber)
- **Reviews** (id, courseId, userId, rating, comment, createdAt)
- **Coupons** (id, code, discountPercentage, expiryDate, usageLimit)

Use Supabase PostgreSQL database for scalability and reliability.

## **FOLLOW-UP INSTRUCTIONS FOR LOVABLE AI**

After the initial generation, use these follow-up prompts to refine:

### **Follow-up Prompt 1: Styling & Branding**
"Apply Cyber Dravida cybersecurity-themed branding throughout. Make the dark theme more prominent with tech-forward colors (teal, cyan, dark gray). Add glowing effects on buttons and hover states. Include a professional logo placeholder area. Make it look modern and trustworthy for tech professionals."

### **Follow-up Prompt 2: Payment Integration**
"Integrate Stripe payment gateway in the checkout page. Add support for all major credit/debit cards. Include test mode indicators. Add order confirmation flow with receipt generation."

### **Follow-up Prompt 3: Database Setup**
"Connect this LMS to Supabase PostgreSQL database. Set up all required tables (Users, Courses, Enrollments, Payments, Certificates, Reviews). Create proper relationships and indexes. Implement Row Level Security (RLS) for data privacy."

### **Follow-up Prompt 4: Authentication System**
"Implement complete user authentication with email/password signup, login, password reset. Add JWT token management. Protect admin routes with role-based access control. Implement session management."

### **Follow-up Prompt 5: Email Notifications**
"Set up email notifications using SendGrid or similar service. Create transactional email templates for: welcome emails, course purchase confirmations, course updates, certificate completions, password resets."

### **Follow-up Prompt 6: Admin Dashboard**
"Create an admin-only dashboard for course management. Add ability to create/edit/delete courses, manage modules and lessons, view analytics, manage students, and monitor payments and revenue."

## **TIPS FOR BEST RESULTS**

1. **Be Specific**: Describe exactly what you want each page to look like
2. **Use Visuals**: If you have competitor websites or design references, mention them
3. **Iterative Approach**: Start with the main prompt, then refine with follow-up prompts
4. **Testing**: After generation, thoroughly test all payment flows and user journeys
5. **Content Preparation**: Prepare your courses content (videos, modules, descriptions) while building
6. **Branding Assets**: Have logo and color specifications ready
7. **Ask for Explanations**: If Lovable generates code you don't understand, ask it to explain
